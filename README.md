# Make Thumbnail
<br/>

- mov, mp4 한개의 영상에서 프레임을 추출하고 원하는 사이즈로 썸네일 만드는 모듈
- OpenCV 사용
<br/>

## How to use

```
Enter video path : /path/video.mp4
Enter save path : /save_directory_path/

Ex. Frame Info
		frame : 217
		width : 1920
		height : 1080
		fps : 24.0

select frame : thumbnail을 만들 int{FRAME} 입력
Enter size : int{WIDTH} int{HEIGHT} 입력 
```
